QR Mesh Chat

Peer-to-peer communication using QR codes as the transport layer

How to Use

  1. Open this page on two devices (phone + laptop, two phones, etc.)
  2. Allow camera access when prompted
  3. Point each device's camera at the other's QR code
  4. Wait for peer discovery - devices will automatically connect
  5. Type messages in the chat box - they'll be queued until connected
  6. Once connected, all queued messages are sent automatically

Built with QR Mesh - TCP-like protocol for peer-to-peer communication over QR codes
